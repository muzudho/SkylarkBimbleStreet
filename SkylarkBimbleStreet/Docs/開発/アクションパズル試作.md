# アクションパズル試作

記録日: 2026-06-10

## 概要

`Game1.cs` に、外部アセットなしで動く 1画面アクションパズル試作を実装した。

## ゲーム内容

- 青いプレイヤーを動かす。
- 黄色い宝石をすべて集める。
- 宝石をすべて集めると、右下の緑の出口がクリア条件になる。
- 赤い障害物に当たるとスタート位置へ戻る。
- `R` または Xbox コントローラー Start でリトライ。
- `Esc` または Xbox コントローラー Back で終了。

## 実装方針

- 内部解像度は `1920x1080`。
- 実ウィンドウにはレターボックス付きで拡大縮小して描画する。
- 外部画像やフォントを使わず、1x1 テクスチャを拡大して矩形だけで描画する。
- 状態表示はウィンドウタイトルと簡易 HUD で行う。
- WASD、矢印キー、Xbox コントローラー左スティックに対応する。

## 確認

```powershell
dotnet build .\SkylarkBimbleStreet\SkylarkBimbleStreet.csproj
```

結果: 成功。警告 0、エラー 0。
