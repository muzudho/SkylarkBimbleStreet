# ゲームをプレイする

## 起動方法

リポジトリーのルートで次のコマンドを実行する。

```powershell
dotnet run --project .\SkylarkBimbleStreet\SkylarkBimbleStreet.csproj
```

## ゲームの目標

![ゲーム画面](images/gameplay-prototype.png)

この画像は開発中の画面イメージです。

- 青いプレイヤーを動かす。
- 切符の破片を3つ集めると、出口が開く。
- 宝石は任意収集物。取った分はステージセレクトのカードに残る。
- 開いた出口に入ると次のステージへ進む。
- 赤い障害物に当たるとスタート位置へ戻る。
- 切符の破片、宝石、出口、障害物は、色だけでなく形でも区別できるようにしている。

## 操作

- 移動: `WASD` / 矢印キー / Xbox コントローラー左スティック
- ポーズ: `Enter` / Xbox コントローラー Start
- リトライ: `R` / Xbox コントローラー Start
- 終了: `Esc` / Xbox コントローラー Back

## 表示設定

![ステージセレクト画面](images/stage-select-prototype.png)

この画像は開発中のステージセレクト画面イメージです。

- ポーズメニューの4枚目のカードで、ゲーム全体の色パレットを切り替えられる。
- ステージセレクトでは、アプリ起動中にクリアしたステージにチェック印が付く。
- 各ステージカードには、壁、切符の破片、宝石、出口、障害物、開始位置のミニマップ風プレビューが表示される。
- カード下部には、過去に取れた最大数の宝石と、通過や無傷などの達成済みバッジが表示される。
- 左右移動時は選択カードが中央へ戻るようにカード列がスクロールする。カード拡大は中央へ戻ったあとだけ、補間付きで滑らかに行う。左右移動は非ループで、初期表示では Stage 1 が中央で選択される。
- ステージセレクト画面とクリア画面にも色パレットが反映される。紙吹雪やランク宝石など一部の演出色は綺麗さ優先で残している。
