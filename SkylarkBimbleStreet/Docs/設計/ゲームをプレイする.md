# ゲームをプレイする

## 起動方法

リポジトリーのルートで次のコマンドを実行する。

```powershell
dotnet run --project .\SkylarkBimbleStreet\SkylarkBimbleStreet.csproj
```

## ゲームの目標

![ゲーム画面](images/gameplay-prototype.png)

この画像は開発中の画面イメージです。

- 青いプレイヤーを動かす。
- 黄色い宝石をすべて集める。
- 右下の緑の出口に入るとクリア。
- 赤い障害物に当たるとスタート位置へ戻る。

## 操作

- 移動: `WASD` / 矢印キー / Xbox コントローラー左スティック
- ポーズ: `Enter` / Xbox コントローラー Start
- リトライ: `R` / Xbox コントローラー Start
- 終了: `Esc` / Xbox コントローラー Back

## 表示設定

- ポーズメニューの4枚目のカードで、ゲーム全体の色パレットを切り替えられる。
- ステージセレクト画面とクリア画面にも色パレットが反映される。紙吹雪やランク宝石など一部の演出色は綺麗さ優先で残している。
