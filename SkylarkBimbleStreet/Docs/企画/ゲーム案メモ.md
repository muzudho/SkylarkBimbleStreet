# ゲーム案メモ

記録日: 2026-06-10

## 前提

これは確定要件ではなく、初期要件を元にしたゲーム案のメモ。

重視する条件:

- 1プレイ 1分から3分で遊べる。
- 即売会で試遊しやすい。
- 無料素材または簡易自作素材で成立しやすい。
- MonoGame 未経験でも段階的に作れる。
- 230時間から460時間の中に学習時間も含めて完成を狙える。
- Windows 11 / デスクトップ PC / ノート PC 試遊に向く。
- 技術を剥がしても残る遊びの核がある。プレイヤーの判断、練習、上達を説明できる。

## 有力候補

### 1画面アクションパズル

1920x1080 の1画面内で完結するゲーム。プレイヤーを動かして、敵、弾、床、スイッチ、ゴールなどを扱う。

例:

- 60秒以内に部屋から脱出する。
- 毎回配置が少し変わる。
- 1ステージ1分で終わる。

向いている理由:

- 即売会試遊に強い。
- 無料素材でも作りやすい。
- MonoGame の基本である描画、入力、当たり判定、音、ステージ管理を学べる。
- ステージ追加で拡張しやすい。

### 短時間スコアアタック

1分から3分で終わる、避ける、集める、壊す系のゲーム。

例:

- 夜の商店街で光るアイテムを集める。
- 制限時間内にできるだけ配達する。
- 障害物を避けながらスコアを稼ぐ。

向いている理由:

- ゲーム内容が未定でもプロトタイプを作りやすい。
- セーブはハイスコアだけで済む。
- Xbox コントローラー、キーボード両対応にしやすい。
- 即売会でプレイヤー交代しやすい。

## その他の候補

### 小さなローグライト風ミニゲーム

毎回ランダムに部屋やイベントが出て、3分で1周するゲーム。

例:

- 3分だけ探索する。
- 3つの選択肢を選びながら進む。
- 最後にスコアや称号が出る。

注意:

- 素材量を抑えつつリプレイ性を出せる。
- ランダム生成やバランス調整で難度が少し上がる。

### 日本語テキスト中心の選択式ゲーム

短い選択と結果で遊ぶゲーム。ノベルゲームほど大きくしない。

例:

- 即売会で3分だけ店番する。
- ひばりが丘近くの不思議な通りを歩く。
- 選択肢でスコアやエンディングが変わる。

注意:

- 日本語のみの方針と相性がよい。
- 素材が少なくても成立する。
- ゲームとしての操作感は弱くなりやすい。

### 盤面系ミニゲーム

グリッド上でキャラやブロックを動かすゲーム。

例:

- 荷物を押してゴールに置く。
- 敵の視界を避けて進む。
- 毎ターン1手ずつ動く。

注意:

- 実装が整理しやすい。
- 無料素材や自作ドットでも成立する。
- 派手さは後から足す形になる。

## いま一番現実的な方向

現時点では、`1画面アクションパズル` または `短時間スコアアタック` が最有力。

理由:

- まず完成形まで持っていきやすい。
- MonoGame の基礎を学ぶ題材としてちょうどよい。
- 試遊時間の条件に合う。
- 素材コストを抑えやすい。

## 仮の外枠

- ジャンル: 1画面アクションパズル、または短時間スコアアタック。
- 1プレイ: 60秒から180秒。
- 画面: 1920x1080 基準。
- 入力: Xbox コントローラー優先、キーボード対応。
- 言語: 日本語。
- 素材: 無料素材または簡易自作素材。
- セーブ: ハイスコア、設定、試遊用リセット。
- 完成目標: 2026年11月頭。
