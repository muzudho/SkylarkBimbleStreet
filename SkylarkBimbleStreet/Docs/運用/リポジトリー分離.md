# リポジトリー分離


## 運用ルール- `ShogiTournamentSystemAnalyzer` と `SkylarkBimbleStreet` は別リポジトリ、別プロジェクトとして完全に分けて扱う。
- Codex が `SkylarkBimbleStreet` の作業をするときは、IDE 側のパス `E:\github.com\muzudho\SkylarkBimbleStreet` を明示して扱う。
- Codex のシェル cwd が `E:\github.com\muzudho\ShogiTournamentSystemAnalyzer`になっていても、`SkylarkBimbleStreet` の作業では cwd に惑わされない。
- `ShogiTournamentSystemAnalyzer` は、明示的に依頼があるまで触らない。
- `SkylarkBimbleStreet` の作業依頼では、必要に応じて作業ディレクトリを `E:\github.com\muzudho\SkylarkBimbleStreet` と明示する。
