# Docs

SkylarkBimbleStreet の開発メモと運用メモの入口。

## 最初に見るもの

- `続きはここから.md`: 次回作業を再開するときの短い引き継ぎ。
- `設計/ゲームをプレイする.md`: 起動方法と操作説明。
- `開発/開発日誌.md`: 日付ごとの開発ライナーノーツ。
- `運用/プロジェクト前提.md`: 作成者、チーム、プロジェクトの前提。
- `運用/メンバー紹介.md`: むずでょさんと Codex の役割、得意不得意。
- `運用/Codex作業ルール.md`: Codex が編集してよい場所、編集しない場所。
- `設計/初期要件.md`: 画面サイズ、開発 PC、未決定事項などの初期要件。
- `企画/ゲーム案メモ.md`: 確定前のゲーム案。
- `開発/アクションパズル試作.md`: 最初の動く試作の内容と確認結果。
- `調査/AIエージェントと素材制作.md`: 他AIや生成AIを素材制作に使えるかの調査。

## フォルダー

- `企画`: ゲーム案、企画案、確定前の提案。
- `設計`: 安定した要件、仕様、ユーザー向け説明。
- `開発`: 実装メモ、調査メモ、動作確認メモ、開発日誌。
- `調査`: 未確定の調査メモ。
- `運用`: リポジトリ運用やドキュメント運用のルール。
