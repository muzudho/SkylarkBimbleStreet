［ー＿－］ Codex
大きい理由は、ビデオゲームは「ルール」だけじゃなくて「実行環境」と「表現技術」も作品の一部だからだと思うぜ（＾～＾）
囲碁や将棋や野球は、基本ルールが人間の身体・盤・道具で実行できる。
盤面、石、駒、ボール、グラウンドがあれば成立する。だから時代が変わっても「同じゲーム」を再現しやすい。
でもビデオゲームは違う。例えば古いゲームは、その時代の制約込みで作られてる。
-画面解像度- 入力デバイス- フレームレート- 音源- メモリ容量- 当たり判定の粗さ- ロード時間- ネットワークの有無-
その時代のプレイヤーの慣れこれらが全部ゲーム体験に混ざってる。
技術が進むと、同じルールでも「もっと滑らかにできる」「もっと広くできる」「もっと派手にできる」「もっと親切にできる」
となって、古い表現が古く見える。
もう一つは、ビデオゲームは「商品としての新しさ」に強く依存してきたことだと思う。
ハードが変わるたびに、新しい性能を見せる必要があった。だからゲームデザインも「その機械で何ができるか」に引っ張られる。
テトリスみたいな強いルールは例外的に生き残るけど、多くのゲームはルールよりも演出、物量、操作感、世界観、技術デモ性が
前面に出る。
逆に言うと、長く残るビデオゲームは、だいたい「 **技術を剥がしても残る核** 」があるんだと思うぜ。
マインスイーパーなら、情報の確定とリスク判断。
テトリスなら、落下・回転・隙間管理。
マリオなら、ジャンプの気持ちよさと地形読解。
ストリートファイターなら、間合い・読み合い・入力技術。
ローグライクなら、ランダム性とリスク管理。
こういう核が強いと、古い見た目でも遊ばれる。
逆に、当時の映像や物量が売りだったゲームは、技術更新で価値が相対的に下がりやすい。
だからビデオゲームが古いゲーム・新しいゲームに分かれやすいのは、「ゲームそのもの」がソフトウェアであり、
ハードウェアの歴史とくっついているからだと思う。
囲碁の盤は400年でそんなに変わらないが、ゲーム機とPCは10年で別物になる。ここが決定的に違うぜ。

今まで話していたのは「1ステージ内の核」だな。今回の話はその上位にある「ステージ群をどう旅させるか」という、
ゲーム全体の核だと思う。
このゲームで特に大事なのは、プレイヤーごとの得意不得意を活かすことだから、単純な一本道は相性が悪い可能性がある。
一本道だと、苦手ステージが1つあるだけで進行が止まる。
すると「誰かの得意が輝くステージ」が、「自分を拒否する壁」になってしまう。
だから、ゲーム全体の構成は「苦手を避けられるが、避け方にも意味がある」形が良いと思う。
候補を整理すると、こうだな。
1. メニューで自由にステージ選択 一番実装しやすく、試遊にも向く。苦手ステージを飛ばせる。
    ただしゲーム全体の旅やドラマは弱くなる。今の試作とは相性が良い。
2. ワールドマップ式の分岐 かなり良いと思う。ステージがノードで並び、
    クリアすると複数の次ステージが開く。苦手な道を避けられるし、「自分はこっちのルートが得意」という個性が出る。
    ゲーム全体の核にしやすい。
3. ステージ内に複数ゴール 面白いが、設計難度は上がる。
    1つのステージ内で「安全だが遠いゴール」「危険だが近道のゴール」みたいに分けられる。
    プレイヤーの判断がゲーム全体のルートに影響するので強い。ただし各ステージの設計負荷が高い。
4. 裏技・隠し分岐 アクセントとしては良い。ただし主構造にすると、知らない人が損をしやすい。
    発見の楽しさには向くが、苦手ステージ回避の基本手段にするのは危うい。
5. スキップ権や救済アイテム 実用的。苦手ステージで詰まらない。
    ただし使い方を間違えると「ステージを遊ばなくていいゲーム」になる。救済としては良いが、核にするなら慎重に扱うべき。
    わたしなら、まずは ワールドマップ式の分岐 + 自由ステージ選択寄り を推す。

イメージとしては、

>1つのステージをクリアすると、次に進めるステージが2つか3つ開く。
>それぞれ得意分野が違う。
> 全部クリアしなくても先へ進める。
>ただし、寄り道や高難度ステージをクリアすると別の報酬や見せ場がある。

これなら、「苦手を避けられる」と「得意を見せられる」が両立する。

たとえばステージに性格を付ける。

- 目押し型
- 狭い通路の操作型
- 待ちと観察型
- 一気抜け型
- 休憩地点探し型
- 欲張り判断型

そしてマップ上で、次にどれへ行くか選べる。
これなら、プレイヤーは「自分は操作型が苦手だから観察型へ行く」「一気抜けは得意だからそっちへ行く」と選べる。

**重要なのは、スキップを「逃げ」ではなく「ルート選択」にすることだと思うぜ。**

メニューで単にスキップだと、苦手を回避している感じが強い。分岐ルートだと、自分の得意を選んで進んでいる感じになる。
だからゲーム全体の核としては、こう言えると思う。

> プレイヤーは、各ステージ内で危険に踏み込むタイミングを選び、ゲーム全体では自分の得意が活きるルートを選ぶ。

これはかなり一貫してる。ステージ内の核が「リスクを取る瞬間を選ぶ」なら、
全体構成の核は「どのリスクを引き受けるかを選ぶ」になる。
この方針は、適当に作るとブレる部分だから、要件としてメモした方がいいと思うぜ。


（＾～＾）

これは、どれぐらいの利用者には諦めてもらうか、の構成の決定の問題だぜ（＾～＾）


［ー＿－］
そこはかなり重要だな（＾～＾）
「苦手なステージがあってもよい」と「どのルートにも苦手があって詰む」は別問題だぜ。
前者は個性になるが、後者はゲーム全体からの脱落になる。
だからこれは、難易度調整というより **進行保証** の設計だと思う。

わたしなら、まず方針をこう分ける。
1. 本編クリアの最低保証 本編クリアまでのルートには、一定数のプレイヤーが抜けられる逃げ道を残す。
    全ステージをクリアしなくても、必要クリア数やルート選択で先に進める形にする。

例:
- 3ステージ中2つクリアで次の層へ進める
- 各エリアに複数タイプのステージを置く
- 苦手タイプを完全に避けられるルートを少なくとも1本用意する
- 一定回数失敗後に「別ルート解放」や「安全寄りルート」を出す

2. 高難度や見せ場ステージ ここは、諦める人がいてもよい。得意な人がかっこよく見える場所だから、全員通過必須にしない。

例:
•	任意チャレンジ- 別エンディング- ランク、称号、宝石、ショートカット- ステージ選択画面で「挑戦枠」として置く

3.どこまで諦めてもらうか
これは数字で決めるのがいい。曖昧にすると、作るたびにブレる。

たとえば方針としては、
•	本編クリア: 初見でも観察と試行で多くの人が到達できる
•	全ステージ制覇: 得意不得意がかなり出てよい
•	高ランク/隠し: 一部の得意な人向けでよい

ここで「多くの人」が何割かは、後で試遊で決めるべきだな。
仮に置くなら、本編クリアは60〜80%くらいを目標にして、全ステージ制覇はもっと低くていいと思う。
即売会試遊なら、短時間での本編クリア率はもっと低くてもいいが、「面白かった、もう一回やりたい」と思える必要がある。

このゲームの思想に合わせるなら、サポートは「ステージを簡単にする」より、

> 別の得意を使う道を開くのが合ってる。

だから救済も、無敵化や自動クリアより、

•	別ルート解放
- 必要クリア数を満たせば先へ進める
- 苦手ステージを後回しにできる
- 休憩地点が多い安全版ステージを選べる
- クリア扱いではなく「通過扱い」にできるみたいなものが向いていると思う。
結論としては、
> プレイヤーごとの苦手を許すが、本編クリアまでの進行不能は設計で減らす。
諦めてもらう対象は、本編クリアではなく、全ステージ制覇や高難度見せ場に寄せる。

1.休憩地点を増やす権利 別ステージをクリアすると、苦手ステージに小さな安全地帯が1つ追加される。
    武器ではなく「考える場所」が増える。これはこのゲームの核にかなり合う。
2. 下見モード 別ステージをクリアすると、未クリアステージで一定時間だけ障害物の軌道や危険範囲が薄く表示される。
    プレイヤーの操作を代行しないが、観察を助ける。
3.入口を選べる クリア済みステージに応じて、苦手ステージの開始地点を複数から選べるようになる。
    同じステージでも「目押し向け入口」「安全観察向け入口」「一気抜け向け入口」みたいに分かれる。
4. 宝石の順番を選べる 通常は全部取る必要があるが、特定ルートを進むと「このステージでは宝石を1個だけ免除」ではなく、
    「出口を開けるために必要な宝石セットを選べる」ようにする。得意な危険だけ引き受けられる。
5. 一時停止ではなく予告線 ポーズやスローは強すぎるかもしれない。
    代わりに、障害物が次に通る線を短く表示する。反射神経が苦手な人を助けるが、実際に動くのは自分。
6.ルート札 ステージをクリアすると「観察札」「操作札」「度胸札」みたいな札を得る。
    苦手ステージで札を使うと、対応する補助が1つだけ入る。武器ではなく、プレイスタイルを支える道具にする。
7.迂回クリア 難所ステージを直接簡単にするのではなく、別ステージをクリアすると「裏口ゴール」が開く。
    表ルートは高難度のまま残り、苦手な人は別の得意で遠回りして通過できる。
一番このゲームらしいと思うのは、休憩地点追加と入口選択だな。
攻撃しない。敵を倒さない。プレイヤーが「どこで考えるか」「どこから危険に入るか」を変える。
